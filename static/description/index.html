Tooltip Many2one Widget

Enhance User Experience with Contextual Information on Hover

The Tooltip Many2one Widget is a powerful utility that allows you to configure and display specific fields from a related record when hovering over a Many2one field. This eliminates the need for users to open the related record to see key details, significantly improving workflow efficiency.

Simply hover over any configured Many2one field in form views, and a clean, informative tooltip will appear with the details you've selected. No more clicks, no more waiting for page loads.

Key Features

  • Per-Model Setup: Define specific fields for each model.
  • Flexible Selection: Choose any field to display.
  • Rich Data: Supports Text, Numbers, Booleans, and Tags.

How to Configure

1. Go to Settings > Technical > Tooltip Many2one Config.
2. Create a new configuration.
3. Select the Model (e.g., Contact).
4. Select the Fields to display.
5. Save and enjoy!
