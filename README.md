# Tooltip Many2one Widget

**Enhance User Experience with Contextual Information on Hover**

The **Tooltip Many2one Widget** is a powerful utility that allows you to configure and display specific fields from a related record when hovering over a Many2one field. This eliminates the need for users to open the related record to see key details, significantly improving workflow efficiency.

## Key Features

*   **Per-Model Setup:** Define specific fields for each model (e.g., Partners, Products, Sale Orders).
*   **Flexible Selection:** Choose any field from the related model to display in the tooltip.
*   **Rich Data Support:** Handles Text, Numbers, Booleans, Selection, and Relational fields (Many2many tags).
*   **Instant Access:** Hover over any configured Many2one field in form views to see details instantly.

## Configuration

1.  Go to **Settings > Technical > Tooltip Many2one Config** (or the custom menu if moved).
2.  Create a new configuration.
3.  Select the **Model** (e.g., Contact).
4.  Select the **Fields** you want to display (e.g., Phone, Email, City).
5.  Save. Now hover over a Many2one field pointing to that model anywhere in Odoo!
